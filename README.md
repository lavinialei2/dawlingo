# Setting up on local machine

First clone the repository using
```
git clone https://github.com/lavinialei2/dawlingo.git
```

Then install all relevant node packages
```
cd dawlingo/my-app
npm install
npm install tone
```

To run the app, use the command
```
npm start
```
